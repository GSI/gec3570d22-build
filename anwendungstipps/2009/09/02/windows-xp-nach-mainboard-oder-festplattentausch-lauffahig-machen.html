Windows XP nach Mainboard- oder Plattentausch lauffähig machen

02 September 2009

Wechselt man bei einem Windows XP System grundlegende Hardware wie das Motherboard, kann dadurch der Betriebssystemstart versagen. Gleiches gilt natürlich, wenn man eine Festplatte mit Windows in einen anderen PC nutzen möchte.

Problembehebung nach dem Umbau

Wenn der Umbau bereits erfolgt ist, kann das Problem wie folgt behoben werden:

  1. Ultimate Boot CD for Windows (UBCD4Win) herunterladen und installieren
  2. UBCD4Win ausführen
  3. Originale Windows XP CD einlegen und als Quellmedium angeben
  4. Quellmedium durch Klick auf “Check” validieren
  5. UBCD4Win brennen
  6. Computer von der UBCD4Win CD starten
  7. Im Boot-Menü Launch “The Ultimate Boot CD for Windows” laden
  8. Zu Start > Programs > Registry Tools > Fix_hdc > Fix hard disk controller navigieren
  9. Mit Taste M eine Aktualisierung der Massenspeicher-Treiber veranlassen
  10. Computer ohne CD neu starten
