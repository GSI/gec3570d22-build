Mittels Firefox Erweiterung Flagfox den Domainbesitzer bestimmen

29 December 2010

Immer wieder werde ich gebeten, herauszufinden, ob bestimmte Domains bereits vergeben sind oder wer aktuell Inhaber ist.

Folgender Linux-Befehl liefert für die gängigsten Top-Level-Domains rasch das Ergebnis:

Der Besuch von Whois-Websites wird dadurch meist überflüssig.

Wer diese Möglichkeit verzichten muss, kann sich im Firefox Web Browser eine andere Einrichtung konfigurieren.

Die "Firefox-Erweiterung “Flagfox”(Beschreibung zum Plugin "Flagfox" auf der offiziellen Website von Mozilla)":https://addons.mozilla.org/en-US/firefox/addon/5791/ fügt der Adresszeile eine kleine Flagge hinzu. Sie zeigt an, in welchem Land die gerade geöffnete Website gehostet wird.

Klickt man mit der rechten Maustaste auf diese Flagge, erscheint in einem Kontext der Menüpunkt “Whois”. Wählt man diesen, wird der Whois-Dienst der Website DomainTools.com geöffnet.

Für Kaufanfragen wendet man sich in der Regel an den administrativen Kontakt (ADMIN-C).

Manchmal sind die Informationen allerdings unvollständig. In diesem Fall findet sich oft ein Vermerk, welcher darauf hinweist, wo man den gesamten Whois-Eintrag abrufen kann ( “For complete domain details go to:” ).

Zusammenfassung der einzelnen Schritte

Einmalig

  1. Flagfox-Erweiterung für Firefox installieren
  2. Firefox neu starten

Für jede abzufragende Domain

  1. Die gewünschte Website/Domain besuchen
  2. Mit der_ rechten Maustaste_ auf die Flagfox-Flagge klicken
  3. “Whois” anklicken
  4. Überprüfen man den korrekten Whois-Datensatz gefunden hat (“Whois Record For GEWÜNSCHTEDOMAIN”); Falls nicht, gibt man die korrekte Domain nochmals im Feld “Search Whois Records” ein.
  5. Die entsprechenden_ Kontaktdaten (z.B. ADMIN-C) ansehen_ oder den Verweis zu den “complete domain details” folgen und sie dort auslesen.

Verwendet man einen anderen Browser, oder möchte man auf Flagfox verzichten, kann man natürlich auch direkt whois.domaintools.com besuchen und die Domain direkt eingeben.
